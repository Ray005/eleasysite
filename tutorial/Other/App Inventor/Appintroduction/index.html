    App inventor2网页版¶

    AI2网页版安装¶

    网页版是较为轻量级的版本，这里使用PC模拟器+网页的开发方式

    1.访问网站并安装PC端程序¶

    • 可以在界面中找到并下载安装aistart（PC的模拟器软件）

    • 或者这里下载：（下载速度肯较慢，你可以使用百度搜索并下载）MIT_Appinventor_Tools_2.3.0 (~80 MB)

    Note

    注意：这里使用的是广州电教馆搭建的app inventor开发服务器

    在哪里能够找到下载地址？

    连接-->模拟器，等待报错在帮助中下载aistart软件，这会重定向到MIT的官方网站

    2.查看AI伴侣版本并下载¶

    在网站app.gzjkw.net上使用可以在帮助中查看所需的ai伴侣版本，下载好： 例如看到：

    1
2
    Companion Version 2.47
[Download URL: http://app.gzjkw.net/companions/MITAI2Companion.apk]

    则下载这个AI伴侣的apk文件，下载好后不要动

    3.在模拟器中卸载旧的AI伴侣¶

    4.使用Dos命令提示符安装新版本的AI伴侣¶

    • 在这样做之前需要将下载好的APK文件复制到安装目录：C:\Program Files(×86)\AppInventor\commands- for-appinventor

    • 然后打开DOS命令提示符，cd进入安装目录(注意需要使用“”），执行
    1
    adb install MITAl2Companion.apk

    就在模拟器里安装了对应版本的Al伴侣了。
