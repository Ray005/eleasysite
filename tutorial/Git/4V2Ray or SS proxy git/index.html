    4.使用V2Ray或SSR代理git¶

    使用V2Ray或SSR均可为git代理，关键是在gitbash中设置代理地址，注意找到代理的本地端口号 * V2Ray 默认10808 * SS/SSR 默认1080 * 佛跳墙 在设置中查看,如19181

    问答：¶

    1. 你正在使用哪个版本的 V2Ray？（如果服务器和客户端使用了不同版本，请注明）

      • 3.29

    2. 你的使用场景是什么？

      • 比如使用 Chrome 通过 Socks/* VMess 代理观看 YouTube 视频。
